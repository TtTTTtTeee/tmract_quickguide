TMR Flooded Roads & DRFA Activations

Quick Guide

Introduction

This web application is mainly focused on the information of flooded roads and Queensland and Disaster Recovery Funding Arrangements (DRFA) activations. It allows users to view both current and historical flood data, filter information based on various criteria, and access detailed information about affected road networks.

Fig. 1: Overview of the TMR Flooded Roads & DRFA Activations application

Navigation

Main Tabs

At the top of the application, you'll find three main navigation buttons:

  • Map: The default view showing the geographical representation of flooded roads
  • Table: Displays the data in tabular format for easier analysis
  • Quick Guide: Opens this guide with instructions on how to use the application

Fig. 2: Main navigation tabs in the header

Flood Data Toggle

Below the header, you can toggle between two views:

  • Historic Flooded: Displays historical flood data records
  • Live Flooded: Shows current flooding situations (active by default)

Fig. 3: Toggle between Historic and Live flood data

Map Interface

Search Function

To find specific locations:

  • Use the search bar in the top-left corner to find specific addresses or places
  • Click the magnifying glass icon to initiate the search

Fig. 4: Search bar for finding specific locations

Map Controls

On the right side of the map, you'll find navigation controls:

  • Zoom in/out buttons (+/-)
  • Home button to reset the map view
  • Other tools for map manipulation and orientation

Fig. 5: Map navigation and zoom controls

Filter Options

The left panel provides comprehensive filtering capabilities to refine the data displayed on the map.

Fig. 6: Filter panel with various filter options

Available filters include:

  • Last updated is between: Select date ranges using the calendar icons
  • Local government: Filter by specific local government areas
  • Locality: Filter by specific localities or suburbs
  • Event priority: Filter by high, medium, or low priority events
  • Road name: Filter by specific road names
TIP After selecting your filters, remember to click the Apply button to update the map view with your criteria.

Legend

The legend panel on the right side explains the map symbols. By default, three layers are visible as follows:

  • TMR Live Flooded Roads: Current flooding with priority indicators (high, medium, low)
  • Local Government Areas: Boundaries of local government regions
  • Active Activations 2024-2025: DRFA activations for the current period

Fig. 7: Legend explaining map symbols and colors

Pie Chart

At the bottom of the screen, you'll find a summary of the current road situation:

  • A pie chart showing the distribution of affected roads
  • Some Key statistics including:
    • Total closures
    • Lanes blocked
    • Lanes affected
    • Roads restricted
    • No blockage
    • N/A status roads

Fig. 8: Status overview with pie chart and statistics

Tips for Effective Use

Filtering Data Use the filter panel to narrow down information to specific areas of interest. This is particularly useful when investigating flooding in specific regions.
Viewing Historical Data Toggle between Historic and Live Flooded views depending on your research needs. Historical data can help identify patterns and recurring flood-prone areas.
Checking Data Currency Always check the Last Updated information to ensure you're viewing current data, especially during active flood events.
Priority Focus Use the different priority indicators in the legend to focus on the most critical areas requiring immediate attention.
Pop-ups - Detailed Information Click on specific road segments or a LGA highlighted with Yellow on the map to open a pop-up for detailed information.

Fig. 9: Example of clicking on an activation layer to view detailed information

Data Sources

The application integrates data from multiple sources including ESRI, TomTom, Garmin, FAO, NOAA, USGS, the Department of Transport and Main Roads, and Queensland Reconstruction Authority.
